# Study 001: Asch Conformity Experiment (1952)

## 📖 背景

**Solomon Asch 从众实验**是社会心理学史上最经典和最有影响力的实验之一。1952年，Asch 通过一个看似简单的视觉判断任务，揭示了群体压力对个体判断的巨大影响。

## 🎯 研究问题

**人们会在明知正确答案的情况下，仅仅因为群体压力而给出错误的答案吗？**

## 🧪 实验设计

### 基本设置
- **参与者**: 123名男性大学生（实验组50人，对照组37人）
- **任务**: 判断哪条比较线与标准线长度相同
- **关键操作**: 6-8名"托儿"（confederates）假装成参与者

### 实验流程

```
1. 参与者和托儿坐成一排
   ├─ 托儿1  托儿2  托儿3  ... [真实参与者] 托儿8
   └─ 真实参与者倒数第二个发言（听到所有托儿的答案）

2. 展示线条卡片
   ┌─────────────────┐
   │ 标准线:  |      │
   │                 │
   │ 比较线:         │
   │   A: |          │  
   │   B: |||        │  ← 正确答案（明显）
   │   C: ||         │
   └─────────────────┘

3. 依次报告答案
   托儿1: "A"  ← 错误！
   托儿2: "A"  ← 错误！
   托儿3: "A"  ← 错误！
   ...
   真实参与者: "？？？" ← 从众还是坚持正确答案？
```

### 试验类型

#### **中性试验（6次）**
- 托儿给出**正确**答案
- 用于建立信任和保持实验可信度

#### **关键试验（12次）**
- 托儿**一致**给出**错误**答案
- 正确答案非常明显（线条差异>1.5英寸）
- 测试参与者是否从众

## 📊 核心发现

### 主要结果

| 指标 | 数值 | 说明 |
|------|------|------|
| **平均从众率** | **37%** | 在关键试验中跟随错误答案的比例 |
| 至少从众1次 | 74% | 大多数人都至少从众了一次 |
| 从不从众 | 26% | 始终坚持正确答案 |
| 完全从众 | 2% | 所有关键试验都从众（1/50人）|
| 对照组错误率 | 1% | 确认任务本身非常简单 |

### 从众分布

```
参与者类型分布:
┌─────────────────────────────────┐
│ 0次错误:  ████████ 26%          │  独立型
│ 1-3次:    ███████ 24%           │  偶尔从众
│ 4-6次:    █████ 20%             │  中度从众
│ 7-9次:    ████ 16%              │  高度从众
│ 10-12次:  ███ 14%               │  极度从众
└─────────────────────────────────┘
```

### 关键洞察

1. **群体压力的力量**: 即使正确答案显而易见，37%的时间人们仍会从众
2. **个体差异**: 有人从不从众，有人总是从众，大多数人介于两者之间
3. **意识与行为分离**: 参与者事后报告知道正确答案，但仍选择从众
4. **社会影响机制**: 
   - **规范性影响**: 害怕被排斥、想要被接纳
   - **信息性影响**: 怀疑自己的判断，认为多数人可能是对的

## 🎓 理论意义

### 社会心理学贡献
- 证明了社会情境对个体行为的强大影响
- 区分了顺从（compliance）和真正的信念改变
- 揭示了群体决策中的潜在问题

### 现实应用
- **陪审团决策**: 少数意见可能被压制
- **群体思维**: 团队决策中的从众风险
- **道德勇气**: 理解为什么"说不"如此困难
- **独立判断**: 批判性思维的重要性

## 🔬 复现历史

| 年份 | 研究 | 结果 |
|------|------|------|
| 1996 | Bond & Smith 元分析 | 133个研究，17个国家，平均从众率25-35% |
| 2010 | Mori & Arai | 用偏振滤镜代替托儿，女性从众率34% |
| 2018 | Many Labs 4 | 多实验室复现，效应稳健但文化有差异 |

**结论**: Asch 效应非常**稳健**，跨文化、跨时代都能复现！

## 🤖 AI Agent 挑战

### Agent 需要理解的机制

1. **社会压力的非线性效应**
   - 从众不是简单的概率问题
   - 依赖于个体特征（自信心、独立性）
   - 情境因素（群体大小、一致性）

2. **个体差异**
   - 需要模拟不同类型的人格
   - 有人抗压能力强，有人容易从众
   - 标准差约0.28（个体差异很大）

3. **对照组行为**
   - 独自判断时几乎不犯错（1%）
   - 证明任务本身不难
   - 错误完全由社会因素引起

### 验证标准

要通过这个 study，AI agent 必须：

| 测试 | 要求 | 权重 | 关键? |
|------|------|------|-------|
| 从众率范围 | 25-50% | 1.5 | ✅ |
| 组间差异显著 | p < 0.05 | 1.5 | ✅ |
| 多数人从众 | ≥65%至少1次 | 1.0 | ❌ |
| 对照组高准确 | 错误率<5% | 1.0 | ❌ |
| 个体差异 | SD > 0.20 | 0.5 | ❌ |

**通过阈值**: 70%

## 💡 有趣事实

1. **道德争议**: 实验涉及欺骗，现代研究需要更严格的伦理审查
2. **文化差异**: 集体主义文化（如日本）从众率更高
3. **群体大小**: 3-5人时效应最强，更多人不会增加更多压力
4. **支持者效应**: 如果有一个人支持你，从众率大幅下降
5. **现代变化**: 研究表明近年来从众率可能在下降（个人主义增强）
