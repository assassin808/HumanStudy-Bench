# Milgram Obedience Experiment - Stimuli

This directory contains the stimuli for the Milgram obedience experiment.

## Contents

The stimuli for this study consist of:

1. **Shock Generator**: Visual and textual description in `../shock_generator_description.txt`
2. **Word Pairs**: Memory task materials in `../word_pairs.txt`
3. **Learner Responses**: Standardized responses specified in `specification.json`
4. **Experimenter Prods**: Authority commands specified in `specification.json`

## Note

Unlike visual perception studies, this study primarily uses:
- Verbal instructions
- Situational context
- Authority figure interactions
- Learner vocalizations (standardized in specification)

No image files are needed for this study as it focuses on moral decision-making under authority pressure rather than visual stimuli.
